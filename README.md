Magist Marketplace Analysis
This project analyzes the Brazilian e-commerce platform Magist to evaluate whether it is suitable for ENIAC, a company selling technology products.
The analysis was conducted using SQL for data exploration and Tableau for visualization.
Main question:
Is Magist a good platform for selling tech products in Brazil?
Conclusion:
Magist shows demand for low- to mid-priced technology products and has reliable delivery performance.
However, it is a general marketplace and not ideal for premium or specialized tech products.
Tools used:
SQL (MySQL), Tableau Public, CSV files
